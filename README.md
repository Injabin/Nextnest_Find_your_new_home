# NextNest: Find Your New Home 🏠

NextNest is a professional JavaFX-based desktop application designed to streamline the process of finding and listing properties. Whether you're looking for a rental or managing property deals, NextNest provides a centralized platform for secure communication, transparent reviews, and easy property management.

---

## 🚀 Overview

NextNest bridges the gap between property owners and seekers. It allows users to browse detailed listings, interact with owners through an integrated chat system, and make informed decisions based on community ratings and comments.

## ✨ Key Features

* **Dynamic Property Listings:** Browse, search, and view properties with detailed descriptions.
* **User Dashboard:** Seamlessly upload and manage your own property listings.
* **Integrated Chat System:** Securely talk with property owners directly within the app to finalize deals.
* **Community Feedback:** View ratings and comments on property posts to ensure quality and trust.
* **Secure Authentication:** Full registration and login system to protect user data and history.
* **Modern UI:** A clean, responsive interface built using JavaFX and CSS.

## 🛠️ Tech Stack

* **Language:** Java 17+
* **UI Framework:** JavaFX (with FXML and CSS)
* **Database:** MySQL (User data, property records, and chat history)
* **Build Tool:** Maven/Gradle
* **Version Control:** Git & GitHub

## 📂 Project Structure

* `src/main/java`: Contains the Java logic, controllers, and database utilities.
* `src/main/resources`: FXML layouts, styling (CSS), and image assets.
* `lib/`: External dependencies and JAR files.
* `.gitignore`: Keeps the repository clean by excluding build artifacts.

## ⚙️ Installation & Setup

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/Injabin/Nextnest_Find_your_new_home.git](https://github.com/Injabin/Nextnest_Find_your_new_home.git)
    ```
2.  **Database Configuration:**
    * Set up a MySQL server.
    * Import the database schema (look for a `.sql` file in the project).
    * Update your credentials in `DatabaseUtil.java`.
3.  **Run the Application:**
    * Open the project in **IntelliJ IDEA**.
    * Ensure the JavaFX SDK is configured in your Project Structure.
    * Run `HelloApplication.java`.
